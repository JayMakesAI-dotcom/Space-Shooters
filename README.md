How to run locally (recommended):
1) Put this folder on a simple local web server (service workers don't run from file://).
   - VS Code: install "Live Server", then right click index.html -> "Open with Live Server"
   - OR: Python: `python3 -m http.server 8000` in this folder
2) Open http://localhost:8000/

Install as an app:
- iOS (Safari): Share -> Add to Home Screen
- Android (Chrome): Menu -> Install app

Hosting:
- GitHub Pages / Netlify / Vercel / any static host
